# Description
A Boilerplate For New Project By fiuyang

# Features
This Boilerplate Project with Node Js, Nest Js, TypeOrm, JWT, Postgresql, Swagger

## Installation
```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start:dev

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Check Docs Swagger
```bash
- http://localhost:3000/api/docs/
```

### Program Will Run
```bash
- http://localhost:3000/
```
